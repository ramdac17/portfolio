SYSTEM OVERVIEW

The PRX - PINOY RUNNER EXTREME Ecosystem is a synchronized full-stack solution built for the extreme running community. It features a high-fidelity Web Dashboard for deep analytics and a React Native Mobile Application for real-time field telemetry.

WEB DOMAIN

    Performance Analytics: Recharts telemetry with real-time "Session Activity" logs.
    Themed Architecture: Cyber-minimalist UI with custom Glitch-Skew CSS and Framer Motion transitions.
    Privacy Governance: Specialized Data Protection Protocol pages ensuring transparency and compliance.

MOBILE DOMAIN

    Native Field Ops: Built with React Native (Expo) for cross-platform reliability.
    Sync Protocol: Seamless data uplink between mobile activity logs and the central database (Railway)
    Tactical UI: Consistent design language using high-contrast "Signature lime cyber tactical minimalist" accents for outdoor visibility.

THE TECHNOLOGY STACK - CORE ENGINES

    Next.js 15: Powering the server-side rendered web application and App Router logic.
    NestJS & Node.js: The heavy-duty backend backbone managing API requests and business logic.
    Swagger: Interactive Command Center for API documentation and endpoint testing.

DATA INFRASTRUCTURE

    PostgreSQL: Relational database for high-integrity activity archival.
    Prisma ORM: Type-safe database management and schema migrations.
    Railway: Primary hosting for the backend database and server service instances.
    Cloudinary: Neural-storage for high-resolution asset management (Fuel, Gear, Mind, Profile, Events and Article imagery).

COMMUNICATION SERVICES

    Brevo & Formspree: Handling automated transactional emails and secure lead ingestion.
    Vercel: Optimized serverless hosting for the frontend web interface.

SECURITY & GOVERNANCE

We treat user data as a tactical asset. The system is hardened with:

    JWT Authentication: Secure session-based access control across Web and Mobile.
    GDPR Alignment: Integrated "Right to be Forgotten" logic and transparent data-usage protocols.
    Encryption: TLS 1.3 standards for all data-in-transit (Uplinks/Downlinks).
    Sovereignty: Users maintain full control over their biological and performance logs.
